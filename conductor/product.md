# Product Definition: Mobile Shop Manager\n\n## Initial Concept\nAn open-source Windows desktop application for mobile shop inventory management, featuring Excel sync, GST billing, and ZPL thermal printing.\n\n## Target Audience\n- **Small-scale Mobile Retailers:** Shop owners in India who need a simple tool for inventory management and GST-compliant billing.\n- **Wholesale Distributors:** Businesses managing large inventories aggregated from multiple sources and suppliers.\n\n## Core Goals\n- **Inventory Unification:** Create a single, "source of truth" dashboard by merging and mapping diverse Excel inventory files from multiple suppliers.\n- **Billing Automation:** Streamline the sales process with automated GST calculations and professional PDF invoice generation.\n- **Precise Labeling:** Enhance physical inventory management using high-speed, accurate ZPL thermal printing for 2-up barcode labels.\n\n## MVP Features (Phase 1)\n- **Dynamic Excel Sync:** Load and merge multiple Excel files with user-defined column mapping (IMEI, Model, Price, etc.).\n- **GST Billing Engine:** Generate tax-inclusive/exclusive invoices with automatic CGST/SGST/IGST calculations.\n- **ZPL Label Designer & Printer:** Native support for Zebra/thermal printers to print barcodes and item details directly from the inventory.\n- **Persistent ID Registry:** Ensure item status (Sold/Returned) remains consistent even when source Excel files are updated or replaced.\n\n## Future Vision\n- **Mobile Companion:** A companion app for real-time stock lookups and barcode scanning using mobile cameras.\n- **Marketplace Integration:** Automatic stock synchronization with major Indian e-commerce platforms like Amazon and Flipkart.\n\n## Value Proposition\nMobile Shop Manager solves the chaos of fragmented supplier data and manual record-keeping. It offers a lightweight, affordable alternative to enterprise ERPs while ensuring professional compliance through automated GST billing and accurate physical stock tagging.
