# Technology Stack: Mobile Shop Manager\n\n## Core Runtime\n- **Language:** Python 3.10+\n- **Execution Environment:** Windows Desktop (Native)\n\n## Frontend / GUI\n- **Framework:** Tkinter (Standard Library)\n- **Theming:** `ttkbootstrap` (Modern Material-style UI components and themes)\n- **Asset Handling:** `Pillow` (PIL) for image and icon management.\n\n## Data & Persistence\n- **Excel Processing:** `pandas` and `openpyxl` for high-performance reading and merging of inventory files.\n- **Local Database:** `TinyDB` for lightweight, JSON-based persistent storage of local data (settings, metadata).\n- **File Monitoring:** `watchdog` for real-time tracking of changes to inventory Excel files.\n\n## Specialized Hardware & Output\n- **Thermal Printing:** `pywin32` (win32print) for sending raw ZPL (Zebra Programming Language) commands directly to thermal label printers.\n- **Billing/Reporting:** `reportlab` for programmatically generating GST-compliant PDF invoices.\n- **Barcode Generation:** `python-barcode` for creating standard barcode formats for retail tagging.\n\n## Packaging & Distribution\n- **Build Tool:** `PyInstaller` for creating standalone, zero-dependency Windows executables (`.exe`).
